# react-web-2022
## Overview
### Frontend:
- Use library ReactJS, SCSS, MUI, Swiper
- Responsive website
- Template Reference: 
    + https://akavo.itech-theme.com/wishlist
    + https://demos.wrappixel.com/premium-admin-templates/react/adminpro-react/main/dashboards/modern
### Backend:
- Use framework Express
- Database: MongooDB
